# CardPreview

카드 컴포넌트는 **카드 번호, 유효 기간, 이름**을 받아 카드 모양으로 미리보기를 띄워줍니다.

카드번호가 4로 시작하는 비자 카드인 경우 비자 아이콘을 띄워줍니다.

카드번호가 51~55로 시작하는 마스터 카드인 경우 비자 아이콘을 띄워줍니다.

<br />
<br />

## Using the Component

```js
<CardPreview
  cardNumbers={cardNumbers}
  date={date}
  ownerName={ownerName}
  cardCompany={cardCompany}
  CVCNumbers={CVCNumbers}
/>
```
